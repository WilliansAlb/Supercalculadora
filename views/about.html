EL PROYECTO

Consiste en una calculadora que mediante la solicitud a una API, devuelve el resultado de las operaciones que se le solicitan, pudiendo ser alguna de las siguientes:
  • Suma
  • Resta
  • Multiplicacion
  • División
  • Potencia
Las herramientas utilizadas para la creación de esta supercalculadora fueron las siguientes:
  • Node.js
  • Modulo Express
  • Modulo Nodemon
  • Modulo EJS
Puedes ver y probar el funcionamiento de la calculadora dando click en IR A CALCULADORA o ver el repositorio de github WilliansAlb
